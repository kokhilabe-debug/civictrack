About CivicTrack

CivicTrack is a frontend-only web application developed using HTML, CSS, JavaScript, and JSON. It demonstrates public infrastructure issue monitoring, priority analysis, map visualization, and analytics.

Technologies Used

HTML, CSS, JavaScript, JSON, Leaflet.js, Chart.js, Git, GitHub, and GitHub Pages.

Version Control Workflow

This module was developed in a separate Git branch, pushed to GitHub, and merged into the main branch to demonstrate Git branching and merging.
